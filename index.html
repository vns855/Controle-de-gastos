Controle de Gastos Pessoais CLI

v1.0.0

Sobre o Projeto

Aplicação de linha de comando em Python para controle de gastos pessoais. Exibe cotação do dólar em tempo real via API pública.

Funcionalidades

  • Adicionar gastos
  • Listar gastos
  • Remover gastos
  • Calcular total
  • Cotação do dólar em tempo real

Como usar

git clone https://github.com/vns855/Controle-de-gastos.git
cd Controle-de-gastos
uv sync --dev
uv run python src/app.py

Exemplo de uso

=== Controle de Gastos Pessoais ===
Cotação do dólar: R$ 5.25
1. Adicionar gasto
2. Listar gastos
3. Remover gasto
4. Mostrar total
0. Sair
